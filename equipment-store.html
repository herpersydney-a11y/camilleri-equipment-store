Equipment Store — Setup
⚙️
🔧 One-time setup required. Do this on your laptop, then share the page URL with workers.
Step 1 — Set up Google Sheet
Go to sheets.google.com → create a new blank spreadsheet. Name it Camilleri Civil Equipment.
Click Extensions → Apps Script.
Delete all existing code. Paste the contents of apps-script-code.js (provided alongside this file).
Press Ctrl+S to save. Then in the function dropdown, select initSheets and click ▶ Run. Approve permissions when asked. This creates the Equipment and Log sheets with default data.
Click Deploy → New deployment. Set type = Web app, Execute as = Me, Who has access = Anyone. Click Deploy. Copy the Web App URL.
